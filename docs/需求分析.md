# 需求分析

## 选做

- 设计导航功能的图形界面，包括地图展示和输出路径展示

- 能够使用课表图形界面方式进行课程管理和查询

## 需求分析

- 用户管理

- 日程管理

- 班级管理

- 地点管理

- 临时事务管理

用户：

- id

- 名称

- 权限

- 密码

班级用户：

- class_id

- user_id

班级：

- id

- 名称

日程：

- id

- 名称

- 地点

- 开始时间

- 结束时间

- 重复方式

- 详情

- 创建者

- 班级_id

- 类型

> 课程、课外活动、个人活动

临时事务：

- id

- 名称

- 地点

- 时间

- 重复方式

- 详情

- 创建者

- 班级

地点：

- id

- 名称

- 坐标

- 类型
